A start at making a First Person View Mod for Plateup (Workshop)

Controls:

F1: Activate
W: Move Forward
A: Strafe Left
S: Move Back
D: Strafe Right
Mouse: Control Camera


Current Features:
- First Person Camera
- Minimap Overhead
- WASD FPS style movement
- Mouse input for camera movement
- Controller Support


Issues:
- Can't access UI

- Basic Skybox applied as some levels are black.
